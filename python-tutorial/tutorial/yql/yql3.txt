Просмотр таблицы
Во всех системах, поддерживаемых YQL, данные находятся в таблицах. Таблицы состоят из именованных колонок, а также могут содержать 0 и более строк.
Пример простой таблицы.
У каждой колонки есть имя и тип, причем все имена колонок в одной таблице не могут повторяться.
Т.к. тип задается у колонки в целом, то и у всех значений в каждой строке таблицы в одной колонке тип будет один и тот же.
В частности, в таблице users , колонка age имеет тип Uint64 (неотрицательное целое число в диапазоне от 0 до 2^64-1), а колонка name имеет тип опциональный String (т.е. может содержать значение NULL).
Запрос для просмотра всех колонок таблицы состоит из указания кластера use hahn , указания списка колонок * после select и имени таблицы после секции from . Т.к. путь к таблице часто содержит символ /, то сам путь указывается в бэктиках.
Запрос для просмотра выбранных колонок таблицы отличается тем, что после select перечисляется список колонок через запятую.
Так как в таблице может быть много строк, то отображается только ее начало. Для явного ограничения числа строк используется секция limit. При этом могут быть выданы произвольные строки таблицы, но не более, чем указано в значении limit.

use hahn;
select * from `home/yql/tutorial/users`;

use hahn;
select name, age from `home/yql/tutorial/users`;

use hahn;
select name, ip from `home/yql/tutorial/ordered_users` limit 5;
